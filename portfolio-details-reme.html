Apliación para almacenar notas o tareas.

Esta apliación te permite almacenar notas o tareas que tienes pendiente realizar con la posibilidad de establecer una alarma para que te lo recuerde.

Registro:

  • Usuario Propio
  • Gmail
  • Facebook

Introducir texto:

  • Mediante el teclado
  • La voz

Todos los datos se almacenarán en una base de datos para que puedas acceder a ellos desde cualquier teléfono solo necesitas tener la aplicación en el teléfono instalada y hacer con tus datos de registro.
